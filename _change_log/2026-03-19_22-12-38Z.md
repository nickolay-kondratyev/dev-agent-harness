Root cause: The Sonar Gradle plugin processes each module independently. With the coverage report path set at the root level (which has no source files), Sonar could not resolve any file references from the Kover XML report, producing 'File X.kt not found in project sources' for all ~140 files. Fix: moved the property to project(':app') { sonar { properties { } } }.

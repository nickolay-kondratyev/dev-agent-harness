## What Changed

Updated  Branch Naming section.

**Before**: When branch slug exceeded the length limit, `TicketShepherdCreator` called `DirectQuickCheapLLM` to compress the title — introducing an LLM call at harness startup.

**After**: Deterministic truncation algorithm:
1. Slugify full title
2. If slug ≤ MAX_SLUG_LENGTH (50 chars) → use as-is
3. If longer → take first K whole words fitting in MAX_SLUG_LENGTH-7 chars, append `-{hash6}` (first 6 hex chars of SHA-1 of full slug)

**Benefits**:
- No LLM call at startup → no startup failure mode from LLM unavailability
- Same ticket always yields the same branch name (deterministic/idempotent)
- Zero latency overhead
- Removes DirectQuickCheapLLM from startup critical path

## What Changed

### New Files
- `CompactionTrigger.kt` — enum with `DONE_BOUNDARY` (extensible for V2 `EMERGENCY_INTERRUPT`)
- `PrivateMdValidator.kt` — validates PRIVATE.md exists and is non-empty after SelfCompacted signal

### Modified Files
- `PartExecutorImpl.kt` — Added `performCompaction()`, `afterDone()` returns `CompactionOutcome` sealed class, nullable handles for both doer and reviewer paths, `DoerSignalResult`/`ReviewerSignalResult` sealed classes for compaction state communication, `respawnAfterCompaction()` for session rotation
- `ShepherdValType.kt` — Added `COMPACTION_TRIGGER` for structured logging
- `PartExecutorImplTest.kt` — 12 new compaction tests (44 total), covering: happy path, healthy context skip, stale context skip, protocol violation, PRIVATE.md missing/empty, timeout, session rotation, threshold boundaries, reviewer PASS compaction failure, crashed session cleanup

### Review Fixes
- Reviewer PASS path now checks `CompactionOutcome` — no silent swallowing of compaction failures
- Crashed signal during compaction now calls `killSession` — no resource leaks
- Temp instruction file cleaned up after compaction

## Changes
- **BranchNameBuilder.kt**: Added `truncateWithHash()`, `buildWordPrefix()`, `sha1Hash6()` helpers. Constants: `HASH_SUFFIX_LENGTH=7`, `MAX_WORD_BUDGET=43`.
- **BranchNameBuilderTest.kt**: Comprehensive test coverage — boundary cases (50/51 chars), single long word fallback, spec example (`c33b35`), determinism, uniqueness.
- **doc/core/git.md**: Updated spec example hash from placeholder `a1b2c3` to actual `c33b35`. Removed implementation gap note.

## Why
Simple character-level truncation could produce identical branch names for tickets with similar title prefixes, breaking uniqueness. The hash suffix ensures different tickets always get different branches.

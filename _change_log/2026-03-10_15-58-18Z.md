## Implementation

- **HarnessServer** interface extending AsgardCloseable with start()/port()/close()
- **KtorHarnessServer**: Ktor CIO engine, binds port 0, writes port to $HOME/.chainsaw_agent_harness/server/port.txt
- **AgentRequest** interface + 4 typed data classes (AgentDoneRequest, AgentQuestionRequest, AgentFailedRequest, AgentStatusRequest)
- **PortFileManager**: plain class for port file write/delete with DEFAULT_PATH companion constant
- Common inline handleAgentRequest<T> for DRY endpoint handlers
- Ktor 3.1.1 + content negotiation with Jackson

## Dependencies Added

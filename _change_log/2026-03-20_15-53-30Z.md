## What
Added `StraightforwardWorkflowE2EIntegTest` — a black-box end-to-end test that:
- Creates a temporary git repo with ticket, CLAUDE.md, and workflow config
- Runs `app/build/install/app/bin/app` via ProcessBuilder with GLM env vars
- Asserts harness behavior (exit code, .ai_out, feature branch), NOT agent coding quality
- 15-minute timeout, diagnostic log tail on failure, delta-based tmux cleanup

Also updated `test_with_integ.sh` to build the binary (`installDist`) before integration tests.

## Changes

### New InstructionSection subtypes (InstructionSection.kt)
- **FeedbackItem(feedbackContent, currentPath, isOptional)** — delegates to InstructionRenderers.feedbackItemInstructions(). SKIPPED only valid for optional__ prefix.
- **StructuredFeedbackFormat** — returns InstructionText.REVIEWER_FEEDBACK_FORMAT wrapped in `<critical_to_keep_through_compaction>` tags.
- **FeedbackWritingInstructions** — returns InstructionText.FEEDBACK_WRITING_INSTRUCTIONS (no compaction tags).
- **FeedbackDirectorySection(dir, heading, filenamePrefix?)** — globs `*.md` files, renders under heading, returns null for empty/missing dirs.

### Supporting Changes
- Added `ProtocolVocabulary.COMPACTION_SURVIVAL_TAG` constant.
- Updated `doc/core/ContextForAgentProvider.md` to reflect filenamePrefix param.

### Tests
- Comprehensive BDD tests for all 4 subtypes including edge cases (empty dir, non-existent dir, non-.md files, prefix filtering, optional vs required feedback items).

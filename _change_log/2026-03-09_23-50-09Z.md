## AiOutputStructure (com.glassthought.shepherd.core.filestructure)

Manages the `.ai_out/${branch}/` directory layout for the Chainsaw agent harness.

### API
- 13 pure path resolution methods (harnessPrivateDir, sharedDir, planDir, planningRoleDir, planningPublicMd, planningPrivateMd, planningSessionIdsDir, phaseRoleDir, sessionIdsDir, publicMd, privateMd, sharedContextMd, locationsFile)
- `ensureStructure(branch, parts, planningRoles)` — idempotent directory creation
- `Part(name, roles)` data class for structured part definitions
- Fail-fast: validates repoRoot is a directory, all string params are non-blank

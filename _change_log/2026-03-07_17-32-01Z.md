## Changes

### New: `InteractiveProcessRunner`
Adds `InteractiveProcessRunner` class to `app/` module that uses `ProcessBuilder.inheritIO()` — wires child process stdin/stdout/stderr directly to the JVM terminal. Enables running interactive CLI tools (e.g., `claude`) from within Kotlin code.

**API:**
```kotlin
val runner = InteractiveProcessRunner(outFactory)
val result = runner.runInteractive("claude")  // blocks until user exits
println("Exit code: ${result.exitCode}")
```

### Returns `InteractiveProcessResult`
Descriptive data class with `exitCode: Int` and `interrupted: Boolean`.

### Updated `App.kt`
Demonstrates usage — launches `claude` interactively, returns when user exits.

### Tests
6 unit tests covering construction, happy/unhappy exit codes, `InteractiveProcessResult` fields.

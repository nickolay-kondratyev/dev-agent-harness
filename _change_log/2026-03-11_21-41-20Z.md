## Changes

### New interfaces in DirectLLM.kt
- `DirectQuickCheapLLM : DirectLLM` — fast/cheap tasks (branch name compression). V1: GLM-4.7-Flash
- `DirectMediumLLM : DirectLLM` — mid-tier (reserved, no V1 callers yet)
- `DirectBudgetHighLLM : DirectLLM` — expensive tasks (FailedToConvergeUseCase). V1: GLM-5

### New production files
- `GlmAnthropicCompatibleApi` (internal delegate) — shared HTTP logic, eliminates duplication
- `GLMQuickCheapApi` — thin wrapper implementing DirectQuickCheapLLM, delegates to GlmAnthropicCompatibleApi
- `GLMHighestTierApi` now implements `DirectBudgetHighLLM` (was bare DirectLLM)

### Initializer refactoring
- `DirectLlmInfra`: replaced single `glmDirectLLM: DirectLLM` with `quickCheap: DirectQuickCheapLLM` + `budgetHigh: DirectBudgetHighLLM`
- Split factory into `createGLMQuickCheapLLM` + `createGLMBudgetHighLLM` with shared `resolveApiToken()` helper
- Removed misleading `getConfigurationObject()`/`Config`/`GLMDirectLLMConfig` — replaced with `Constants.resolveMaxTokens()`

### Constants additions
- `DIRECT_LLM_API_MODEL_NAME.GLM_QUICK_CHEAP = "glm-4.7-flash"`
- `Z_AI_API.ANTHROPIC_API_VERSION = "2023-06-01"` (replaces magic string)
- `resolveMaxTokens(): Int` focused helper

### Tests
- `GlmApiHttpContractTestHelper`: shared parameterised test helper (14 HTTP contract tests)
- `GLMQuickCheapApiTest`: new, 16 tests (delegates to shared helper + type check)
- `GLMHighestTierApiTest`: added type-check test (now 16 tests)
- 150 total tests, 0 failures

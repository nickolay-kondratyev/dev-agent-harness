## Wingman Session ID Tracker

New `Wingman` interface (`suspend fun resolveSessionId(guid: String): String`) and `ClaudeCodeWingman` implementation at `com.glassthought.chainsaw.core.wingman`.

### How It Works
- Recursively scans `~/.claude/projects/.../*.jsonl` for a GUID marker
- Extracts session ID from matched filename (UUID without `.jsonl` extension)
- Fail-fast: throws `IllegalStateException` on 0 or >1 matches

### Implementation Details
- Constructor-injected `claudeProjectsDir: Path` for testability
- `withContext(Dispatchers.IO)` for blocking file I/O
- `Files.walk().use {}` for proper stream cleanup
- Structured logging with `Out`/`Val`/`ValType`

### Testing
- 7 BDD unit tests: happy path, no match, ambiguous match, nested dirs, non-JSONL filtering

## Changes

- **GlmAnthropicCompatibleApi**: now implements both DirectBudgetHighLLM and DirectQuickCheapLLM; tier is captured by model name at construction, not type.
- **GlmDirectLlmFactory**: removed anonymous object wrappers; createBudgetHighLLM/createQuickCheapLLM delegate to new public create() method returning GlmAnthropicCompatibleApi.
- **ContextInitializer**: replaced two private factory methods with single createGlmApi(model, outFactory, httpClient). Common config params (maxTokens, apiEndpoint, apiToken) now resolved once, eliminating divergence risk.

All tests pass (32 for GLM tier tests).

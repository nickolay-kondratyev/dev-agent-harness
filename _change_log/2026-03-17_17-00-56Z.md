## Key Design Decision
Single-signal (HTTP callbacks) chosen over dual-signal (callbacks + file polling) because:
1. In-session agent death (TMUX alive, agent dead) is rare
2. Acceptable to wait ~33 min instead of ~8 min for this edge case
3. Eliminates significant spec complexity (net -127 lines)

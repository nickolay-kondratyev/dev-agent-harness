## Spec changes

- **AutoRecoveryByAgentUseCase.md**: Marked V2 DEFERRED, design preserved for future

## Why
- Fail-fast is MORE robust than 20-min recovery agent timeout
- Recovery agents have their own failure modes (crash, timeout, wrong fix)
- Index.lock handles ~80% of git failures (Pareto)
- Clear error message lets human diagnose the actual issue

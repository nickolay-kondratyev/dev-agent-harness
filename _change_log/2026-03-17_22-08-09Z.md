## Changes

Deleted GLMHighestTierApi.kt and GLMQuickCheapApi.kt — each was a 42-line class with zero business logic (pure delegation to GlmAnthropicCompatibleApi).

Created GlmDirectLlmFactory (internal object) with:
- createBudgetHighLLM(...): DirectBudgetHighLLM
- createQuickCheapLLM(...): DirectQuickCheapLLM
- private buildDelegate(...) — single GlmAnthropicCompatibleApi construction site

Updated ContextInitializer to call factory methods. Updated all 3 test files (GLMHighestTierApiTest, GLMQuickCheapApiTest, GLMHighestTierApiIntegTest) to use GlmDirectLlmFactory. All 14 HTTP contract tests + tier interface checks preserved and passing.

Result: ~80 lines → ~42 lines, 2 files eliminated, GlmAnthropicCompatibleApi constructor wiring in 1 place (was 3).

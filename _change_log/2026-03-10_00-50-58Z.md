## Role Catalog Loader

Implements auto-discovery and loading of agent role definitions from `$CHAINSAW_AGENTS_DIR`.

### New Files
- `app/src/main/kotlin/com/glassthought/shepherd/core/rolecatalog/RoleDefinition.kt`
- `app/src/main/kotlin/com/glassthought/shepherd/core/rolecatalog/RoleCatalogLoader.kt`
- `app/src/test/kotlin/com/glassthought/shepherd/core/rolecatalog/RoleCatalogLoaderTest.kt`
- Test resources in `app/src/test/resources/com/glassthought/shepherd/core/rolecatalog/`

### Design
- Mirrors TicketParser pattern: interface + companion factory + Impl
- Reuses `YamlFrontmatterParser` from ticket package
- Flat directory scan (maxDepth=1) via `Files.walk().use {}`
- All file I/O on `Dispatchers.IO`
- Fail-fast on missing description, empty/non-existent directory

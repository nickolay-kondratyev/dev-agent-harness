## Components

### BranchNameBuilder (object)
- Pure stateless utility for building branch names from TicketData
- Slugify algorithm: lowercase → replace non-alphanumeric with hyphens → collapse → trim → truncate at 60 chars
- Anchor point: ap.THL21SyZzJhzInG2m4zl2.E

### GitBranchManager (interface + GitBranchManagerImpl)
- createAndCheckout(branchName): wraps git checkout -b
- getCurrentBranch(): wraps git rev-parse --abbrev-ref HEAD
- Uses ProcessRunner from asgardCore, optional workingDir via git -C
- Structured logging with Out/Val/ValType

### Testing
- 17 BranchNameBuilder unit tests (slugify edge cases, build format, validation)
- 4 GitBranchManager integration tests (gated with isIntegTestEnabled)
- All tests passing

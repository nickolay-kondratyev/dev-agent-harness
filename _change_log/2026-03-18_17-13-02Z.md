## Changes
- Deleted `AgentRole` enum and `UnifiedInstructionRequest` data class
- Added `ExecutionContext` data class for shared doer+reviewer fields (composition)
- Added sealed `AgentInstructionRequest` with 4 typed subtypes per spec (ref.ap.9HksYVzl1KkR9E1L2x8Tx.E)
- Interface method: `assembleInstructions(request: AgentInstructionRequest): Path` (single param)
- Added `privateMdSection()` to impl — reads `${sub_part}/private/PRIVATE.md`, silently skips if absent
- PrivateMd inserted at position 2 (after RoleDefinition) in all 4 role plans
- Removed all `requireNotNull` runtime guards (compile-time via sealed types)
- Removed spec implementation-gap note

## Tests
- 4 requireNotNull guard tests removed (states now unrepresentable at compile time)
- 3 PrivateMd tests added (included when present, silently skipped when absent)
- All existing tests updated to use sealed subtypes

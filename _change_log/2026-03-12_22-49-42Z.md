## Changes

### Removed
- **DirectMediumLLM** interface: spec says 'add mid-tier when a use case emerges'; code added it unused ('Reserved — no V1 callers yet')
- **Environment** class + test: not in spec, never used in production code, anti-pattern (production code aware of isTest flag)
- **Constants.CLAUDE_CODE.DEFAULT_MODEL**: was only used for the wrong coupling below

### Fixed
- **ClaudeCodeAgentSessionIdResolver model coupling**: model was baked into constructor (hardcoded to 'sonnet' at init time). Spec says model comes from sub-part config at spawn time per sub-part. Moved to `resolveSessionId(guid, model)` parameter — AgentSessionIdResolver interface updated accordingly.
- **TmuxSessionManager.createSession**: was taking `String` instead of `TmuxStartCommand`, defeating the type safety wrapper. Updated to take `TmuxStartCommand`.

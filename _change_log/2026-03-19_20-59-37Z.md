## Health-Aware Await Loop (ref.ap.QCjutDexa2UBDaKB3jTcF.E)

### Changes
- **AgentFacadeImpl**: 3 new constructor deps (AckedPayloadSender, AgentUnresponsiveUseCase, QaDrainer). sendPayloadAndAwaitSignal now implements full health monitoring loop with HealthAwareAwaitLoop inner class.
- **QaDrainAndDeliverUseCase**: Extracted `fun interface QaDrainer` for DIP compliance.
- **ShepherdValType**: Added CALLBACK_AGE, TIMEOUT_THRESHOLD, STALE_DURATION, TMUX_SESSION_NAME for structured health logging.
- **AgentFacadeImplTest**: 15+ new BDD tests with virtual time (TestClock + kotlinx-coroutines-test advanceTimeBy).

### Flow
1. Deliver payload via AckedPayloadSender (crash on ACK timeout)
2. Health loop polls every healthCheckInterval:
   - Signal completed → return
   - Q&A pending → drain & deliver, skip health checks
   - lastActivityTimestamp fresh → continue
   - Stale > normalActivity → ping via AgentUnresponsiveUseCase
   - No activity after pingResponse → crash declaration

### Review iteration
- Added check() assertion on AgentUnresponsiveUseCase.handle() return for NO_ACTIVITY_TIMEOUT
- Extracted QaDrainer fun interface (was incorrectly using open class)

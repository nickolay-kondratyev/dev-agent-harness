## Problem\nAgents spawned in TMUX got exit code 127 (command not found) when calling callback_shepherd.signal.sh because there was no validation that the callback scripts directory existed.\n\n## Fix\n- Created CallbackScriptsDir class with validated() and unvalidated() factory methods\n- Updated ClaudeCodeAdapter to accept CallbackScriptsDir instead of String\n- Updated ContextInitializerImpl to return CallbackScriptsDir.validated()\n- Added CallbackScriptsDirTest with 10 test cases

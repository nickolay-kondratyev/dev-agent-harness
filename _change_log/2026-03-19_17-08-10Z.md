## Changes

- Added `failedToConvergeIterationIncrement: Int = 2` to `HarnessTimeoutConfig`
- Created `UserInputReader` fun interface + `DefaultUserInputReader` (injectable stdin abstraction)
- Created `FailedToConvergeUseCase` fun interface + `FailedToConvergeUseCaseImpl`
  - Binary y/N prompt via ConsoleOutput
  - Only 'y'/'Y' accepted (with trim), all else returns false
- 9 BDD unit tests covering: y, Y, N, empty, null, whitespace-trimmed input, prompt formatting, custom increment

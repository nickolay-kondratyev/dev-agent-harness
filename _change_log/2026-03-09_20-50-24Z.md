## Root Cause

`Constants.Z_AI_API.CHAT_COMPLETIONS_ENDPOINT` was `https://api.z.ai/api/anthropic` (base URL only). The Anthropic Messages API requires the full path `/v1/messages`.

## Fix

Updated constant to `https://api.z.ai/api/anthropic/v1/messages` in `Constants.kt`.

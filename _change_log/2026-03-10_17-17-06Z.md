## What Changed
- **settings.gradle.kts**: Removed `includeBuild("submodules/thorg-root/source/libraries/kotlin-mp")` composite build block
- **app/build.gradle.kts**: Added `mavenLocal()` to repositories (before mavenCentral)
- **build.gradle.kts** (root): Added `publishAsgardToMavenLocal` and `checkAsgardInMavenLocal` Gradle tasks
- **submodule buildlogic**: Added `id("maven-publish")` to both KMP and JVM convention plugins; explicit publishing block for JVM
- **thorgKotlinMP.build.gradle.kts**: Added `publishAsgardLibsToMavenLocal` aggregate task
- **Documentation**: Updated THORG_ROOT from required to optional (only needed for explicit publish)

## Before vs After
- **Before**: Every `./gradlew :app:build` required THORG_ROOT; composite build recompiled asgard libs from source
- **After**: `./gradlew :app:build` works without THORG_ROOT; asgard libs resolved from ~/.m2 local maven

## Usage
```bash
# One-time publish (or after asgard changes)
export THORG_ROOT=$PWD/submodules/thorg-root
./gradlew publishAsgardToMavenLocal

# Check status
./gradlew checkAsgardInMavenLocal
```
